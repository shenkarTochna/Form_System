Controllers for form system.
Methods in SE final project.

Create forms using the following controllers: 


-Lable 
-TextBox
-ComboBox
-CheckList
-RadioList
-NumericBox
-Button
-MessageBox

All Controllers support the following options :

 Show();
 Hide();
 SetForeground(ForegroundColor color);
 SetBackground(BackgroundColor color);
 SetBorder(BorderType border);
